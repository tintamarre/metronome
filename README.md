# Metronome & Tuner

A web-based practice tool for musicians combining a feature-rich metronome and a chromatic/guitar tuner.

## Metronome

- **Tempo range**: 40–208 BPM with classical tempo markings (Largo, Andante, Allegro, Presto...)
- **Time signatures**: 1–8 beats per measure
- **8 sound presets**: Pendulum, Mechanical, Bip, Classic, Wood Block, Digital, Soft, Deep
- **Accent first beat**: Optional emphasis on the downbeat
- **Beat visualization**: Animated display showing current beat position and bar count

## Tuner

- **Chromatic mode**: Detects any note with real-time frequency and cents deviation display
- **Guitar mode**: 5 tuning presets (Standard, Drop D, Half Step Down, Open G, DADGAD) with auto string detection
- **Visual feedback**: Needle gauge for pitch accuracy, color-coded tuning status (green = in tune)
- **Signal meter**: Shows microphone input level

## Interface

- Light/dark theme toggle
- Tab navigation between metronome and tuner
- Responsive design for mobile and desktop

---

Built with Vue 3, TypeScript, and Vite.
